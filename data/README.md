# Data Folder

This folder is reserved for local documentation or small non-sensitive sample files.

## Notes
This project relies on local research data files that are **not included** in the public repository.

The main pipeline expects input files such as:

- `outputs_cds/data/top50/ve1.csv`
- `outputs_cds/data/top50/adj.npz`

These files should be stored locally and are excluded from version control.
